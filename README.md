## Why

We needed a way to manage attendees and we wanted to provide a great developer experience as well. 
## Tooling

- TypeScript
- GraphQL via Apollo
- React
- React-Router
- Styled Components
- Immer
- Passport
- Mongo with Mongoose and Typegoose
- Koa
